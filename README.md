# SlamApp
This project is a web-based task management application designed for remote teams, following the Scrum methodology. The frontend is built with HTML, CSS, and Bootstrap, utilizing EJS templates for dynamic content rendering. The backend is powered by Node.js, Express.js, and MongoDB, providing a robust and scalable foundation for managing tasks efficiently in a remote work environment.
